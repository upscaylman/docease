# Template Word Exemple

Ce fichier décrit la structure attendue du template Word.

## Fichier requis
Vous devez créer un fichier `template_principal.docx` dans ce dossier.

## Structure du Template

Le template Word doit utiliser cette syntaxe Docxtemplater :

---

OBJET : {contexte}

Paris, le {date}

{nom_destinataire},

{texte_ia}

Points importants à retenir :
{points_importants}

Je reste à votre disposition pour toute information complémentaire.

Cordialement,
[Votre nom]

---
Document généré le {date_complete} à {heure}
Email destinataire : {email_destinataire}

---

## Variables disponibles

- {nom_destinataire} : Nom du destinataire
- {contexte} : Contexte/sujet du document
- {points_importants} : Points importants mentionnés
- {texte_ia} : Texte généré par l'IA (2-3 paragraphes)
- {date} : Date au format dd/MM/yyyy
- {date_complete} : Date complète avec jour
- {heure} : Heure de génération
- {email_destinataire} : Email du destinataire

## Instructions

1. Créez un nouveau document Word
2. Copiez le texte ci-dessus dans Word
3. Remplacez les variables entre accolades par le format Word
4. Formatez le document (police, espacement, etc.)
5. Sauvegardez comme `template_principal.docx` dans ce dossier

Pour plus de détails, voir README.md dans ce dossier.
